BioStamp3

Plug into the world’s most advanced wearable sensor

Collect research-grade physiological data for almost any application you can imagine. Start prototyping today with the BioStamp3™ SDK.

Available for Android, iOS, Node.js and Chromium-based web browsers.

Sensor features

  • Supports thousands of sensing configurations

  • Conforms to almost any part of the body

  • Operates continuously for several days

  • Streams data for real-time feedback

  • Water resistant

  • Rechargeable and reusable

  • Easy to use at home

  • Comfortable and unobtrusive during any activity

Sensing capabilities

Motion

  • 3-axis gyroscope, accelerometer and compass up to 562 Hz

AFE

  • Standalone ECG up to 4 kHz
  • Synchronized Photoplethysmogram (PPG) and ECG up to 1 kHz

Environment

  • Temperature with range of –10°C to 100°C, accurate to ±2°C
  • Piezoresistive pressure sensor (altimeter) with range of 260 to 1260 hPa

Bioimpedance

  • Electrodermal activity (EDA)

Applications

  • Virtual clinical trials in a variety of domains, from motion disorders to diseases of the central nervous system
  • Remote patient diagnostics/telemedicine
  • Lifestyle and activity tracking
  • Advanced research
  • Algorithm development

Development platforms

  • Android
  • iOS
  • Node.js
  • Chromium-based web browsers (Google Chrome and Microsoft Edge)

Developer Resources

  • The BioStamp3 Developer Console lets you configure, test and run your sensors right out of the box, with no programming required
  • Our open source libraries and code samples on GitHub make it easy start building your own sensing applications
